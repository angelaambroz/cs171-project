the shape of text

by angela ambroz

spring 2016 (cs171 (harvard extension school))

Hello!

Welcome to the shape of text, a text visualization of several 19th century novel.

This project was built using:

  • d3.js (data visualization)
  • nltk (text processing)
  • The Gutenberg Project (original text)
  • Bootstrap (pretty website)

Full code is available here on GitHub. For a narrative about the project, including decisions, dead-ends and tutorials, see the process book.
